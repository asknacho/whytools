# Installation 


1) apt update && apt upgrade -y


2) apt install git -y


3) git clone https://github.com/Bhai4You/Termux-Gui.git


4) cd Termux-Gui


5) chmod +x requirement.sh Gui.py


6) bash requirement.sh


7) python2 Gui.py


8) Done..!!!
